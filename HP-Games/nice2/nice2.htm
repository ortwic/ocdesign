Hier finden Sie viele nützliche Programme für Nice 2.

 

 

Milkshape 3D Mit diesem Tool können für Nice 2 eigene Autos erstellt werden. Dieses Programm unterstützt das OBJ-Dateiformat, dass zur weiterverarbeitung der Autos benötigt wird. Eigentlich ist es für animierte Figuren gedacht, die in Spielen wie Unreal Tournament, Quake oder Halflife zum Einsatz kommen. Unter dieser Adresse kann es gefunden werden:
http://www.swissquake.ch/chumbalum-soft/ 

Headmax
OBJ-Konverter
Download		 Dieses Programm konvertiert OBJ/Wavefront-Dateien in das Synetic-OIF-Dateiformat. Durch das Setzen verschiedener Prefixe ist es möglich:
  • die Texturfarbe schwarz als transparent zu definieren
  • Spiegelungen hinzuzufügen bzw. zu entfernen
  • auf die Schattierung der 3D-Objekte Einfluß zu nehmen

Milkshape 3D
Tutorial
Download		 Dieses Tutorial entstand aus der Notwendigkeit, eine deutsche Anleitung zum Auto- und Objektbau zu erstellen, die auf die Besonderheiten bei Nice 2 eingeht. Es erklärt, wie: 
  • 3D-Objekte mit MilkShape3D erstellt werden
  • welche Objekte für ein Nice2-Auto benötigt werden
  • was man unbedingt beachten sollte um Probleme zu vermeiden

Texturentpacker
Download
(ohne TIF-Formate)		 Der Texturentpacker decodiert die SYN-Dateien der Fahrzeuge zu TGA-Dateien. So kann man eigene Texturen erstellen oder heruntergeladene einsetzen. Es gibt auch eine Version, die alle Texturen der Autos im TIF-Format enthält, was das Ändern erheblich erleichtert. Da die Datei 2,58 MB groß ist, steht hier nur der Link zur Homepage von Synetic.
http://www.synetic.de

Strecken Auf dieser Seite finden Sie alles Rund um's Streckenbauen. Dies ist allerdings eine umständliche Sache, da das ganze Spiel in ein spezielles Verzeichnis installiert werden muss, damit die neuen Strecken richtig funktionieren. Um Strecken bauen zu können, wird zudem ein extra Editor für jedes Scenario gebraucht. Die Seite von Autistic Cucoo's hat alles, was dazu gebraucht wird, mit einem ausführlichen Tutorial.
