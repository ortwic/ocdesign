Desktop Notizenverwalter (für Active Desktop)
Bei Diesem Projekt handelt es sich um kein einzelnes JavaScript mehr, sondern um ein Konstrukt vieler JavaScript-Funktionen, die zu einer Art Anwendung zusammen geführt worden.
Wie der Titel schon sagt, kann diese "Anwendung" Ihre Notizen verwalten. Sie können dynamisch Notizzettel anlegen, bearbeiten und löschen. Mit dem Potential, das in der Idee steckt, sind noch viele weiteren Erweiterungen und Einstellungsmöglichkeiten geplant.

Es ist zu 100% vorgesehen, diese "Anwendung" offline zu benutzen. Den besten Nutzen erhalten Sie, indem Sie es als Active Desktop Element in Ihre Windows Umgebung einbetten. Aufgrund dieser Absicht, ist das Programm nur auf dem Internet Explorer nutzbar, weil Windows bekanntlich auf diesem seit Version 4 aufbaut. (also natürlich auch der Active Desktop)
In Windows XP ist die Einrichtung des Active Desktop's etwas versteckt. Eine kurze Navigation:
  • Rechtsklick auf den Desktop
  • Kontextmenüeintrag Eigenschaften wählen
  • Auf der Registerkarte Hintergrund den Button Desktop anpassen anklicken
  • Dort die Registerkarte Web auswählen
  • Die Kontrolbox Webinhalte auf dem Desktop anzeigen muss aktiviert sein
  • Auf die Schaltfläche Neu klicken
  • Mit einem Klick auf Durchsuchen brauchen Sie nur noch den Pfad der heruntergeladenen Datei anzugeben
Ich möchte nicht unerwähnt lassen, dass viele Ideen und Konzepte zu Problemlösungen von diversen Seiten und anderen Scripten stammen. Die Links sind in der "Anwendung" zu finden.

Ab Version 0.3.0 wurde das Script komplett erneuert. Das Hauptfeature ist die Verwaltung der Notizen mittels einer speziellen Klasse. Hierdurch ergab sich endlich die Möglichkeit, die Haltbarkeitszeit für die Cookies festzulegen. Weiterhin kamen grafische Effekte hinzu, wie Transparenz und Untermenüs für das Kontextmenü, sowie die Darstellung von 3D-Schatten. Ferner gibt es ein eigenes Kontextmenü für die Textbearbeitung mit einigen Formatierungshilfen.
